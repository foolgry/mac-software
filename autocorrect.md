## autocorrect

中文写作工具，能够格式化中文文本

### url
https://github.com/huacnlee/autocorrect

### install
```
curl -sSL https://git.io/JcGER | bash
```

### usage

```
autocorrect text.txt
你好 Hello 世界

autocorrect --fix text.txt
autocorrect --fix zh-CN.yml
autocorrect --fix
```
